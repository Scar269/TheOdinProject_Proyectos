Proyectos del curso TheOdinProject

Formulario de Registro

Formulario de registro diseñado con HTML y CSS que simula
una página moderna de creación de cuenta. Incluye
distintos campos de entrada (nombre, correo, contraseña, etc.)
con estilos personalizados y validaciones visuales mediante colores.

Dashboard Grid

Dashboard completo diseñado con HTML, CSS y Grid.
Incluye sidebar con iconos, header con búsqueda y usuario,
cards de proyectos con iconos de acción y panel derecho
